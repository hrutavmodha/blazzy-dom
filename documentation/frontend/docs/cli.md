The Blazzy CLI scaffolds new project of Blazzy DOM quickly with pre-configured templates.

## Install

```bash
npx blazzy-cli my-app
```

You will be prompted to choose a template:

- **JavaScript**
- **TypeScript**

Next steps are printed after scaffolding:

```bash
cd my-app
npm install
npm start
```
